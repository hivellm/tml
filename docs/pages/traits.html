Copy

behavior Copy

Marker behavior for types that can be trivially copied (bitwise copy).

All primitive types (I32, F64, Bool, etc.) implicitly implement Copy. Copy types don't need explicit cloning - assignment copies the value.

Example

// I32 is Copy, so this works:
let x: I32 = 5
let y: I32 = x  // x is copied, both are usable
